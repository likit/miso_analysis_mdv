Miso analysis for Marek's Disease Project
=========================================

A pipeline for running miso analysis on RNAseq data from MDV project

##Prerequisite software
* MISO/0.4.9
* BEDTools/2.17.0
* TopHat/2.0.9
* Bowtie2/2.1.0
* SciPy/1.13.0 and NumPy/1.8.0
* NetworkX/1.6
* SAMTools/0.1.18
